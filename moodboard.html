📐 Princípios
📦

Pacotinho é Real

Envelope de figurinha, não caixa genérica. Textura de papel, linha de corte, formato de pacote Panini.

🎮

UI de Jogo

Bordas ornamentadas, botões 3D com sombra, banners ribbon. Estilo Royal Match / Candy Crush, não Material Design.

✨

Feedback Rico

Toda ação = resposta visual. Abrir pacote = animação de rasgar. Figurinha nova = confete + glow.

🏟️

Identidade Copa

Verde gramado + dourado + vinho. Cada pixel grita futebol. Bandeiras, troféu, estádio.

💎

Raridade Visual

Especial = prata + glow. Rara = roxa + shimmer. Lendária = dourada + pulso. PARECER especial.

📊

Progresso = Game

Nada de texto seco. Anéis de XP, barras coloridas, porcentagens que mudam de cor.
